Capabilities

Dual-Loop Intelligence

Seamlessly transitions between autonomous exploration and social collaboration.

Idle Loop

Autonomous Voyager Protocol that analyzes inventory, health, and tech tree to gather resources and progress independently.

Social Proxy

Instantly pauses autonomous tasks when a player speaks, shifting focus to assist, converse, or collaborate in real-time.

Self-Healing Skills

Drafts modular JavaScript skills, tests them in-game, captures errors, fixes code inline, and saves verified scripts for future use.

Gemini 1.5 Pro

Powered by Google's most capable reasoning model for complex decision-making, code generation, and contextual understanding.

Mineflayer Native

Built on the robust Mineflayer framework for deep Minecraft integration, supporting versions 1.18 through 1.20+.

Customizable Persona

Modify the system instruction to change tone, priority hierarchy, and behavioral quirks to match your play style.

System Design

Architected for
Autonomy

Jules operates on a sophisticated dual-loop engine. The central orchestrator manages environmental perception, skill execution, and state transitions — all while maintaining persistent memory of learned abilities.

1

Environmental Perception

Builds a JSON state payload with coordinates, health, biome, nearby blocks, and entities every few seconds.

2

Skill Execution System

Searches existing skills, drafts new ones when needed, executes code, and self-heals on runtime errors.

3

Iterative Learning

Verified skills are saved to disk as reusable JavaScript modules, creating a lifelong skill library.

Quick Start

Get Running in Minutes

01

Clone & Install

bash 
git clone https://github.com/fiozxr/jules.git
cd jules
npm install
02

Configure Environment

.env 
GEMINI_API_KEY=your_gemini_api_key_here
BOT_USERNAME=AI_Brother
HOST=localhost
PORT=25565
03

Launch

bash 
node index.js

Requires Node.js v16+, a Minecraft Java server, and a Google Gemini API Key.
